HexEdit 3.2 Release Notes
=========================

HexEdit 3.2 contains several useful enhancements to the properties dialog as well as improved large file support.

Major new features include support for ZLIB compression/decompression (as used in many file formats, communications protocols etc) including complete support for all ZLIB options.

- new find (offset) options
- template menu
- template improvements (string switch) and bug fixes
- new templates IFF, BMP ...
- background image, about box


See the What's New section of the help for more details.
