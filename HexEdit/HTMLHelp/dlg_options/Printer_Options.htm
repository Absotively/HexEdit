Printer Options Page

 

 

The Printer Options page gives you a great deal of control over how pages are printed.  Not only can you set margins and line-spacing but you can created custom headers and footers with substitutable parameters taken from the following list.

 

 

When printed is replaced with:

&F

The file name

&A

The full path name of the file

&P

The current page number

&D

The current date

&T

The current time

&N

The current date and time in long format

&C

The date and time the file was created

&M

The date and time the file was last modified

Header and Footer Special Parameters and Replacements

 

Header

The header is printed at the top of each page.  It is divided into a left section, a centred section and a right section.  Separate each section by a vertical bar (|).  If you do not supply a vertical bar the header is all left justified.

 

The header can contain any text and/or any of the substitutable parameters in the table above.  The header is displayed within the top margin (see below) so make sure enough room is provided.  You can position the header with the Header (From Edge) control below.

Footer

The footer is similar to the header but is printed at the bottom of each page.  It is divided into a left section, a centred section and a right section.  Separate each section by a vertical bar (|).  If you do not supply a vertical bar all of the footer is left justified.

 

The footer can contain any text and/or any of the substitutable parameters in the table above.  The footer is printed within the bottom margin (see below) so make sure enough room is provided.  You can position it with the Footer (From Edge) control below.

 

Draw border

If selected a line is drawn around the page (not including the header and footer).

Column headings

If selected headings are drawn at the top of each page showing the offset of the column from the address on the left.  The column numbers are printed in hex if hex addresses are being used, else they are decimal (see Decimal Addresses command).

 

Units

The units of measurement for margins and header/footer positioning.  This will default to local conventions (set in the control panel), but you can set it to inches or cm as desired.

 

Single Line Spacing

Lines are spaced as shown on screen.

1.5 Line Spacing

The distance between printed lines is 50% more than single line spacing.

Double Line Spacing

The distance between printed lines is double that of single line spacing.

 

Left Margin

Minimum amount of blank space to leave on the leftt side of the page..

Right Margin

Minimum amount of blank space to leave on the right side of the page.

 

Note that the left and right margins may appear larger than specified here, depending on the font in use and the numbers of columns on screen.  HexEdit will print in the same size font as used on screen unless there is insufficient room in which case it will use the largest font size that fits within the left and right margins.

Top Margin

The amount of space to leave at the top of each page.  If you have specified a header you must leave enough space for it.

Bottom Margin

The amount of space to leave at the bottom of each page.  If you have specified a footer you must leave enough space for it.

 

Header distance from edge

Distance of the page header from the top of the page which allows it to be positioned vertically.  The header must be positioned within the top margin -- if this value is too large then the header will not be printed.

Footer distance from edge

Distance of the page footer from the bottom of the page which allows it to be positioned vertically.  The footer must be positioned within the bottom margin -- if this value is too large then it will not be printed.
