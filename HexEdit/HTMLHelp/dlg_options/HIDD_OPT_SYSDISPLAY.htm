Display Options Page

 

 

The Display options page allows you to change some global options related to the display.

 

Restore main HexEdit window on startup

The main HexEdit window is restored to its position when you last ran the program. For this option to have any effect the settings must be saved (see the Save settings options below) since it is only effective when the HexEdit starts up.

Display Window Tabs

When this options is selected, tabs are displayed at the top or bottom of the main HexEdit window that show all the files and windows currently open.  Each tab shows an icon for the file type and the file name.  To switch to any window just click on its tab.

Window Tabs at Bottom

When this checkbox is on window tabs (see above) are displayed at the bottom of the main HexEdit window.  Otherwise they are displayed at the top.

 

Display hex digits in uppercase

When this option is selected the digits A to F in all hex numbers are displayed in uppercase. This affects all windows, dialogs, the Find Tool, the Hex Address Tool, etc. This option is recommended as there is less chance of confusing certain digits (such as "6” and lowercase “b”), depending somewhat on the font you are using.

Large (block) cursor

A more easily visible cursor (or “caret”) is used to show the current edit position in the active window. When off, the cursor is a thin flashing line to the left of the current edit position. When on the cursor changes to be a much wider block covering the character at the edit position. This is useful for people using a small monitor at high resolution or for people with poor eyesight.

Shadow Cursor in Hex/char Area

When you are editing you current edit position is marked with a flashing cursor.  You can edit in the hex or character area but it is handy sometimes to see what the corresponding value in the other (character or hex) area is.  When this option is on the current value in the other area is indicated also.

 

No Template View

When you create a new file or open a file that you have not opened in HexEdit before this option says to not show the template view.  Note that even with this option on it is a simple matter (since HexEdit 3.2) to open the template view using the Split Window command or the Tabbed Window command on the Template Menu.

Split Window

When creating or opening a new file the template is displayed to the left of the normal view in a split window.  This allows you to see the template view and the data view at the same time and even adjust their relative sizes by dragging the splitter.  The template used is that associated with the file extension of the file or the "default" template if there is none.

Tab Window

When creating or opening a new file the template is the tree view is displayed in a separate tabbed window.  You can only see one of the normal view or the tree view at a time but can easily switch between them by clicking on the appropriate tab.  The template used is that associated with the file extension of the file or the "default" template if there is none.

No of fixed array elements

For large arrays (with constant element size) it may be faster to display only the first few elements of arrays.  This setting allows you to change how many elements are displayed..

Default template data format strings

These strings determine how data elements of a template are displayed if the template file does not explicitly specify the format.  For example, you may want to see unsigned integers displayed in hex (%x) rather than decimal (%u).  Refer to C language printf format specifications for more information, and strftime for date formats.
