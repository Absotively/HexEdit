Customize Mouse Page

 

 

Commands can be associated with a double-click of the mouse in three different areas of a HexEdit window: the main hex area, the character area to the right of the hex area (if shown) and the address area to the left of the hex area

 

To customize a double-click event: select the area in the list on the left, select the “Associate double-click event with” radio button and select the command from the list.

 

By default double-clicking in the hex and character areas sets the mark position. Double-clicking in the address area selects the current line. You should reassign these to commands you use frequently. For example, you may like to change the address area double-click to toggle between hex and decimal addresses.

 

View

This lists the three double-click areas that may be customized.

No double-click event

Select this to have the mouse double-click event (in the selected area) do nothing.

Associate double-click event with

Click this to associate a double click (in the area selected in the list on the left) with a particular command, then select the command from the list below.
