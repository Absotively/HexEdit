Keep Times command

 

This command says to keep the same file times (ie those that were in effect when the file was opened) when the file is saved to disk.

 

Normally when you save a file it's time of last modification and time of last access are updated to the current time.  Using this command the times that were in effect when the file was opened are restored when you save the file.  This can be used so that other software or people are unaware that the file has been changed.  For example, this could be used for changing makefile source files without causing unnecessary rebuilds.
