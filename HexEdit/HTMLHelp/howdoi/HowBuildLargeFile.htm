How Do I Build a Large File?

 

There is a 16 Mbyte limit on the clipboard in Windows so it is difficult or impossible to build up a file of more than about 50 Mbytes by copying and pasting large blocks from other files.  To do this you need to make use of the Append Selection to File and Append Same File commands.

 

You will need to put the bits of the file together one at a time in order.  First use the Write Selection to File command to create a new file or Append Selection to File to append to an existing file.  Thence you can select blocks of text and use the Append Same File command to add further blocks to the end of the same file.  Take care with this command, however, as there is no prompt to confirm that you want to append to the file and no way to undo the operation without opening the file and manually removing the bytes.
