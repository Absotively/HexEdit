Customization

 

The Customize dialog allows you to completely tailor the user-interface to your own needs and preferences. You can drag any command or submenu in any of the categories in the Commands page to a toolbar or menu. Similarly the Keyboard page allows you to assign commands to a keyboard shortcut or check what keys are already assigned. You can even assign commands to context (popup) menus and mouse double-click events (see below).

 

The Toolbars page allows you to create and edit toolbars. HexEdit comes with two built-in toolbars, called Tool Bar and Edit Bar, and a menu bar. If you have made a major mistake with your customization, or the menu bar or toolbars are messed up you can use the “Reset” or “Reset All” buttons to restore them to the way they were originally. You may also need to do this if you have installed a new version over the top of an old version of HexEdit and you cannot see the new commands.

 

The search and address tools (that, by default, are on the Edit Bar) can now be resized (new in version 2.0). For example you might work with long search strings and want to make the find tool wider. You can also change toolbar button image by selecting from some supplied images or even use the button image editor to create your own.

 

The Customize Menus page allows you to modify the two mainframe menus (the cut-down menu that appears when there are no files open and the full menu that appears when one or more files are open). You can also modify seven different context menus when you right-click on the following areas

  1. status bar

  2. address area of window display

  3. hex area (unless selected or highlighted)

  4. character area (unless selected or highlighted)

  5. the current selection (in hex or character area)

  6. highlighted bytes (in hex or character area)

  7. bookmarks (in hex or character area)

There is also a special customization feature that allows you to run any command when the three areas of the display (address, hex and character areas) are double-clicked with the mouse. This should be reserved for very often used commands.  By default double-clicking in the hex or character area sets the mark to the byte clicked on, while double-clicking in the address area selects the row of bytes at that address.

 

The customize Tools page allows you to easily set up external programs to be executed from the HexEdit Tools menu. By default this is set up to run Notepad and the Windows Calculator.
