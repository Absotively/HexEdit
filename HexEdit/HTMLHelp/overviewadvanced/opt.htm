Options Dialog

 

Options on the System page: prevent multiple instances of HexEdit from being run, put “Open with HexEdit” on shell shortcut menus, turn on background searches, determine whether backup files are created, and whether the options you have changed are saved when you exit HexEdit. You can also set the options how you want them and save them immediately with the “Save Now” button.

 

The Display page of the Options Dialog allows you to make global changes to the display. You can set HexEdit to always maximize it's main display when it is opened, and display tabs along the top or bottom of the main window to allow you to quickly flip between files.  You can also set whether hex digits are displayed in uppercase (A-F) or lowercase (a-f) wherever hex numbers are displayed or entered. You can also choose whether you want nice looking addresses displayed – that is, thousands separators in decimal addresses or four digit grouping in hex addresses.  There are also options to change the size of the cursor and to have a 2nd "shadow" cursor in the other (hex/character) area.

 

The Windows page allows you to change all of the display settings of the active window.  If there are no files open then this page does not appear.  Most of these options are available from toolbar buttons but a few (such as setting the column grouping size) are not.  This page also allows you to set how new windows are displayed by default using the "Save As Default" button.

 

The colors page allows you to change any of the colors of the active window's display, including background and the color that address and even different ranges of byte values are shown in.  Moreover, you can create complete color schemes that can be easily selected from the "Color Scheme" drop-list of the "Standard Toolbar".

 

The Filters page allows you to configure the drop down list of filename filters that is displayed in the File Open and File Save dialogs.  The Printer Options page allows you to change how pages are printed.
