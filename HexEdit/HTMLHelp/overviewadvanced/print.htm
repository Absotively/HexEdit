Printing

 

If you are acquainted with printing in other Windows programs printing is fairly simple and straightforward. What you see in the window is almost exactly what is printed, but to see how the file would be spread across pages you can use the Print Preview command. You can select the printer to use, and options such as landscape or portrait printing with the Page Setup command.

 

Page Layout

 

The Printer section of the Options Dialog allows setting of page margins, line-spacing, etc.  You can also customize the header and footer displayed on each page, with the current date/time, page number, filename, time file was last modified etc. There are also options to print a box around the actual data (ie, excluding the header and footer) and to print column headers so you can work out the exact address of any byte.  Use the Print Preview command to see exactly how the printed page will look.

 

In versions prior to HexEdit 2.1 printing was always done in the same size font as was displayed on screen.  This meant that, depending on the width of the printer page and the number of columns displayed on screen some of the printing may have been lost off the right side of the display.  This has now been addressed.  Now, if the printed line does not fit between the left and right margins, then a smaller version of the same font is used so that it does fit.

 

Partial Printing

 

It is not unusual to want to print a small part of a binary file. You can do this in HexEdit by printing a range of pages or by printing the selection.

 

To print a range of pages but you must first use Print Preview to find out the numbers of the pages you want to print.  Another problem with printing a range of pages is that the standard Print Dialog in Windows has problems if the total number of pages exceeds 32,767. If you want to print part of a very large file such as a database, this is a problem.

 

A better way to print part of a file is to select the bytes you want to print and use the “Print Selection” option of the Print Dialog.  Also when you print the selection you have the option of not printing duplicate lines (instead the line is printed just once with a count of the number of repeats below) -- this can be very useful if the section you want to print contains large areas of null (zero) bytes.

 

Template Tree View

 

To print (or print preview) the template tree view simply select an item in the tree view and use the File/Print (or File/Print Preview) command as usual.  Note that the tree view is printed exactly as displayed.  You may want to expand or contract different branches before printing.  Also the text is sized to fit across the page so you may need to resize or hide some columns if the text prints too small.
