File Open dialog

 

 

This dialog allows you to select or specify an existing file or files to open.  This dialog is used with the File Open command, the Read File Command and the Append Selection to File command.

 

The folders and files in the current location are displayed. Double-clicking a folder opens it – that is, your current location moves to that folder. To open the folder one level higher (the “parent” of the current folder) click the “Up One Level” button.

 

Only files of the type selected in the “Files of Type” list box are displayed. You can view the files sorted by name, date & time, size etc by clicking the detailed list button and clicking the appropriate column heading to sort by that column. The dialog also allows you to delete (see How Do I Delete a File On Disk?), rename, view properties etc by clicking the file with the right mouse button and selecting from the context menu that appears.

 

Double-clicking a file opens the file. You can open more than one file by shift-clicking to select a contiguous group of files (click the first file with the mouse and hold the Shift key down while clicking the last). You can also select individual files by Control-clicking them. After you have selected all the files you want to open click the Open button.

 

Look In

To see how the current folder fits in the hierarchy on your computer, click the down arrow. To see what's inside a folder, click it.

File Name

Type the filename you want to open. If you type the name of a folder that folder is opened and the contents are displayed. Normally you see all files matching the specification currently selected in the “List Files of Type” box – but if you type a filename with wild-cards here, such as *.DOC, you will only see matching files.

List Files of Type

Select the type of file(s) you want to open.  By default all files (*.*) are displayed, but you can elect to use a different filter by selecting from the drop down list.  You can change the default filter (*.*) and change, add or delete from the filter list by using the Filters page of the Options dialog.

Open as read-only

Select this option if you want to prevent accidental modifications to the file. Note that even without this option the file may still be open read-only if it is open elsewhere or if the disk file has the read-only attribute set.  Also windows on a file may be read-only even if the file is open for changes (see Allow Changes Toggle).

Open sharebale

Select this option to allow other programs to modify the file while it is open in HexEdit.  This option is generally not recommended as the file contents may change unexpectedly and without notice.
