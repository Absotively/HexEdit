File menu commands

 

The File menu offers the following commands:

 

New

Creates a new file.

Open

Opens an existing file.

Close

Closes an opened file.

Save

Saves an opened file using the same file name.

Save As

Saves an opened file to a different file name.

Print

Prints a file, or part thereof.

Print Preview

Displays the file on the screen as it would appear printed.

Print Setup

Selects a printer, connection etc.

Export Motorola S1 Records

Writes the selection as a Motorola S file using S1 records only.

Export Motorola S2 Records

Writes the selection as a Motorola S file using S2 records only.

Export Motorola S3 Records

Writes the selection as a Motorola S file using S3 records only.

Export Intel Hex Records

Writes the selection as an Intel hex format file.

Export Hex Text

Writes the selection as a file containing hex digits.

Import Motorola S Records

Reads a Motorola S record file.

Import Intel Hex

Reads an Intel hex format file.

Import Hex Text

Reads a file with hex text.

Recent Files

Re-opens a recently opened file.

All Recent Files

Displays a list of all recently opened files.

Exit

Exits HexEdit.
