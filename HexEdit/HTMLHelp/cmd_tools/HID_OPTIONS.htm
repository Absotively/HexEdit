Options command

 

This displays the Options dialog, which allows you to change the behaviour of HexEdit.

 

This dialog has the following pages:

System Options Page allows you to change how HexEdit behaves globally

Display Options Page allows you to change how HexEdit looks globally

Window Display Page allows you to change settings for the active window (and defaults)

Macro Options Page allows you to change how macros are played

Colours Options Page allows you to change the colour in which different bytes are displayed

Printer Options Page allows you to set margins, headers etc for printed output

Filters Options Page allows you to set the filters available in File Open/Save dialogs
