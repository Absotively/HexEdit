Advanced Template Creation Tutorial

 

Welcome to the HexEdit Advanced Template Creation Tutorial.

 

Who Should Use this Tutorial?

 

This tutorial is for those who are serious about creating HexEdit templates.  If you just want a simple introduction to creating templates or are not even sure what you want from templates why not try the Basic Template Tutorial first?

 

It should take less than 60 minutes to complete the tutorial.

 

Before You Start...

 

It is assumed that you have a good understanding of the use of HexEdit and know the basics of using and creating templates.  If not try reading Using Templates, Expressions, Editing templates and/or Basic Template Editing Tutorial.

 

What You will Learn

 

In the tutorial you will learn how to create all the template data structures, including optional (IF), variable length arrays (FOR), recursive data structures, end of file "headers", etc.  It also explains how to do validation and improve the usability and appearance of your template.

 

The tutorial does not teach anything about the format of the underlying template (XML) files.  You do not need to know that to create and edit templates.  To find out about the format of template files see Template Files.

 

There are six steps:

1. The Data File Format

2. Creating the Header

3. Creating the Graphical Object Structure

4. Creating the Object Array

5. Creating the Colour Table

6. Validation and Testing

 

At the bottom of each step there is a button, which allows you to move to the next step (and a button to allow you to go backwards).
