Toolbars: Tool Bar command (View menu)

 

Use this command to toggle on or off the display of the “Tool Bar”. The Tool Bar is a default toolbar that has been superceded by the Standard Toolbar (see View Standard Toolbar), but remains for backward compatibility with existing customized toolbars. A check mark appears next to the menu item when the Tool Bar is displayed.

 

See Toolbar for help on using the toolbar.  The Customize Commands page allows you to completely reconfigure the Tool Bar.
