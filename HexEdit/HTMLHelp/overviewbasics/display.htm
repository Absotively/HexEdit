The Display

 

The HexEdit display is divided into two or three areas. The main area in the middle of the window is the “hex area” which contains pairs of hex digits representing the bytes of the file. To the left is the address of the first byte of the row (displayed in hex or decimal). Optionally to the right of the hex area (or instead of it) is the “character area” which shows the corresponding characters (in the current character set).  You can edit the file by changing hex values in the hex area, or by changing characters in the character area.

 

At the top of the HexEdit window is the title bar, menu bar and (typically) toolbars.  At the bottom is the status bar.  There are often one or more modeless dialogs that are available too, such as the Properties dialog, Find dialog and Calculator.

Windows and Files

 

HexEdit allows you to edit multiple files simultaneously, and you can open more than one window on the same file. For example, you might want to make changes to the end of a file, while viewing the start of it. To do this, open two windows for the same file and scroll one to the end of the file and start making the changes. You can even make the other window “read only” so that you don’t accidentally make changes in it.

 

The tabs at the top (or bottom as in the above screen-shot) of the display allow you to quickly see what files and windows you have open.  By clicking on the tab you can quickly change to a different window.

 

Every window in HexEdit has a set of attributes associated with it that can be set independently. When you reopen a file the same attributes are restored, so the window looks exactly like it did when you last worked on it.  These attributes are kept in the "recently used file list" so if you clear the list of recently used files then this information is lost.

 

This information includes the cursor position and how the file is displayed -- the number of display columns (default 16) and the column grouping (default 4), which areas (hex and character) are displayed, what character set to use in the character area (ASCII, EBCDIC, etc), the font, colour scheme and other display options. A useful feature is “AutoFit” mode where the number of display columns is not fixed but is adjusted to fit the width of the window when it is resized.

 

You can also configure a window to use overtype or insert mode, and toggle between read only and modifiable modes.

 

Each window also has an associated “mark”, which allows you to mark a position, return to it, and measure distances from it.  You can also create named bookmarks, which allow you to quickly return to a location in a file (even opening the file if necessary).  Further you can highlight a group or groups of bytes using the highlighter.  By default bookmarks are shown with a blue background and highlighted bytes a yellow background (but this can be adjusted for each colour scheme -- see below).

 

Adjusting the Window Display

 

Most of the window display options can be adjusted using the Standard toolbar.  For example, you can select the character set and colour scheme from drop down lists, quickly increase/decrease the font size, toggle modes etc.  More options are provided on the Format Bar, such as selecting a new font from the drop down list.

 

The “Window” page of the Options dialog allows you to change all these display settings plus others (like the column grouping) which cannot be changed from a toolbar  This page also has a button to save the current settings as the default which is used when a new window is created – ie, when a new file is opened or created.

 

There are four default colour schemes (associated with the four character sets that can be displayed), but you can create your own schemes.  To use a new scheme just select it from the drop-down list on the Standard toolbar.  (You can also associate a scheme with a file type based on file extension).  The scheme determines all colours including background colour, highlighter colour etc.  It also defines groups of byte values which are displayed as different colours -- for example, the ASCII default colour scheme shows ASCII control characters in red.  The "Colours" page of the Options dialog allows you to change the colours of any colour scheme and to create new colour schemes.

 

Character Sets and Support for EBCDIC and Unicode

 

The display of four different character sets is supported (ASCII, ASCII+ANSI, ASCII+IBM/OEM, and EBCDIC).  You can select these from the drop-down list on the Standard toolbar.  The character set in use only affects how characters are displayed in the character area not the hex area.

 

The ANSI character set is an extension of ASCII and hence differences are only apparent when viewing characters with the high bit on.  The IBM/OEM character sets is the set of characters that were used for the video display of the original IBM PC and compatibles -- when displaying using the IBM/OEM character set a different set of fonts is available from the normal Windows (ANSI) fonts.

 

As Windows has no EBCDIC fonts the EBCDIC characters set is supported by translating to an ANSI font.  In EBCDIC mode searches are performed using the EBCDIC character set, and copying/pasting to the clipboard translates to/from ASCII as appropriate.

 

Note that the display of the Unicode character set is not supported for two reasons.  First Unicode characters use two bytes and it is not possible to say which pair of bytes to use. Also Windows 95/98/ME do not support Unicode characters. However, under Windows XP/2000/NT you can view the Unicode character at the cursor position (ie the current and following byte) using the Character Properties Page.  Unicode is also supported in searches and pasting from the clipboard.  

 

Global Options

 

Most display options are controlled separately for each window – you can use a different font etc in each window. There are, however, several global options that change the display in all windows. For example, there is a global option to determine whether hex numbers (ie, digits A to F) are displayed in upper- or lower-case. These are found on the “Display” page of the Options Dialog.
