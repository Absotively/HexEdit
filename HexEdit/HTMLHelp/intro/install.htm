Installing and Removing HexEdit

 

To run HexEdit you need Windows 95 or later, or NT 4 or later including Windows 2000/XP/Vista.  The HTML help system and the template system requires Internet Explorer 5.

 

Installation

  • Before you install HexEdit 3.2 you need to obtain the installation file hexed3_2.msi. It may be distributed in a zip file called something like hexed.zip or hexed3_2.zip. You can find out how to obtain this file from the HexEdit web site at http://hexedit.com.

  • When you double-click the hexed3_1.msi the Microsoft Windows Installer (MSI) program is run.  If you have an older version of MSI you will be prompted to download the appropriate version.

  • Follow the instructions.

  • You may install the following components:
    Program Files     – You need these files if you want to run HexEdit!
    Help Files     – These should be installed if you want to use the on-line help.
    Example Templates – several HexEdit templates for common types of file.
    Skins     – Sample skins that let you change the look of the HexEdit user-interface.

  • If you intend to run HexEdit from a console command line then it needs to be installed in your PATH. You can do this by adding the directory where you installed HexEdit to your PATH.

  • The first time you run HexEdit (and thereafter until you register) you see a dialog reminding you that the version is unregistered.  To register immediately just click the registration web link. To find other ways to register the software click one of the buttons.

  • If HexEdit is corrupt or improperly installed you may see a message such as “HexEdit is not properly installed”. In this case you must manually remove it and reinstall – see Error: HexEdit has not been installed or the FAQ at the HexEdit web site (http://hexedit.com) for details.

  • You can try HexEdit for free for 30 days.  To pay for it go to our registration web page.

Removing HexEdit

  • In Control Panel double click the “Add/Remove Programs” icon.

  • Click HexEdit in the list of files that may be removed and click the Remove button.

  • Click “Yes” if you really want to remove HexEdit. The removal procedure is displayed.

  • After HexEdit has been removed just click the OK button.

  • If HexEdit was corrupted or improperly installed you may not be able to remove it. In this case you may manually remove it – see Error: HexEdit has not been installed for details.

Note: Some things may not be completely removed until you shutdown and restart the system. For example, you may still see HexEdit shown in the Start Menu until you reboot.
