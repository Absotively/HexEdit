Support

 

We’d like to hear from you if you want to report a bug, request an enhancement, or just tell us you like the program.

 

If you have a question about how to use HexEdit please check the on-line help first. If you still can’t work it out you might try the FAQ (frequently asked questions) or the latest bug list at the HexEdit web site (see below). If you still can't work it out post a message on our forums (http://www.hexedit.com/forums.htm).

 

Registering

 

You can register your copy of HexEdit at our registration web page.

 

Email

 

It is very easy to get in contact with us if you are connected to the Internet for email. While you are running HexEdit you can send email using the “Help via Email” command on the Help menu. If you want to send email from a machine where HexEdit is not running, or you can’t get HexEdit to run at all, send email to: support@hexedit.com.

 

Alternatively, you may get a better response by posting to the Forums on our web site.

 

Important notes:

  • To make quite sure my spam filter does not discard your email, please include the word “HexEdit” in the title.

  • Your email address(es) are kept strictly confidential, and will never be given or sold to anyone else.

If you have found a bug include the following:

 

1. A brief description of the problem in the subject prefixed by “HEXEDIT BUG:”

2. Describe the problem as fully as possible in the message body.

3. Try to remember exactly what you did and what happened.

4. List the steps required to reproduce the problem if you can reproduce it.

5. Include the exact text of any error messages from HexEdit or the system.

6. Include the operating system version and the version of HexEdit you are using.

7. Try to think of anything peculiar to your configuration that may be relevant to the problem.

8. Include your email address(es) in case we need to contact you for more information.

 

Web Site

 

If you want to register your copy of HexEdit, have questions on how to use it, want to participate in a discussion group, want to download the latest version or find out about new features, then check out the HexEdit web site at http://www.hexedit.com.

 

To quickly connect to this site while running HexEdit use the “Help via the Web” command on the Help menu. Of course, you have to be connected to the Internet at the time.
