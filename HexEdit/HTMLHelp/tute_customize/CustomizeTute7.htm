Step 7. Miscellaneous Customizations

 

         

 

In the final step of this tutorial we will look at several other simple customizations.

 

 

Automatic Macros

 

By giving macros a special name they can be run automatically.  If you save a macro using a name for a specific file extension then that macro will be run whenever a file with that extension is opened.  For example, if you save a macro to a file called "_EXE.HEM" then whenever an .EXE file is opened in HexEdit the macro is run automatically.

 

This facility can have many uses -- as an example,  for a certain file type you might always want to position the insertion point after the header at the start of file.

 

Another special macro called "AUTOEXEC.HEM" is run whenever you start up HexEdit.

 

To demonstrate we will create a macro for .HEG files that jumps past the header at the start of the file.  The (fictitious) .HEG file format has a 16 byte header.

 

1. Open any file, to allow recording of macro commands that operate on a file.

 

2. Select Edit/Go To which brings up the Calculator, and click the "Hex" radio button if not already in Hex mode.

 

We open the calculator before macro recording is started so that it is not recorded as part of the macro.  We need the calculator open as it is used to jump to address 16 (10 hex) in the file.

 

3. Select Tools/Record Macro (or double-click the "REC" pane on the status bar) to start macro recording.

 

 

3. Enter "10" in the calculator and click to "GO" button.

 

4. Select Tools/Record Macro (or double-click the "REC" pane on the status bar) to stop macro recording.

 

Now we just need to save the macro we have created to disk.  It is given the file name "_HEG.HEM" so that it is run when any .HEG file is opened.  Note that the first character of the file name (before HEG) is an underscore - this denotes that the macro file is special.

 

5. Open the Options dialog (Tools/Options) and select the "Macro" tab.

 

6. Click the "Save Current Macro" button and name it "_HEG".  Add a comment to the macro if you like.

 

HexEdit add the ".HEM" file extension automatically for HexEdit macro files so you do not need to type that.

 

7. Click OK to save the macro then click OK to close the Options dialog.

 

8. Open the file called "TUTORIAL.HEG" which you can find in the folder where HexEdit was installed.

 

9. Verify that the cursor (insertion point) jumps to the 16th byte.

Default Display

 

HexEdit remembers the display options for all files you have ever opened.  When you open a file that you have had open before all the formatting options are restored exactly as they were when you last used it.  Even the cursor is restored to where it was when you closed the file.

 

But when you open a file that you have never opened before the default display settings are used.  These are also used when you create a new file in HexEdit.  You can easily change the default settings as shown here.

 

1. Open any file.  We need a file open to modify (and preview) display settings.

 

2. Open the Options dialog (Tools/Options) and go to the Window page.

 

The Window page of the Options dialog is not shown if you skipped step 1 and do not currently have a file open.

 

3. To see the current default display settings click the "Reset to Default" button.

 

4. Change some of the options.  For example, click the Font button to change the default font used.

 

The factory default font is "Courier", which is not a TrueType font.  This font is chosen as it is a fast, nice-looking monospaced (non-proportional) font available on all versions of Windows.  If you prefer a True Type font (eg because they scale better) then Lucida Console is a nice monospaced font available on recent versions of Windows.

 

See the Window Display Options page for information on the different fields.

 

To preview the display settings changes at any time just click the "Apply" button to see how they look in the active file.

 

5. Click the "Save as Default" button.

 

6. Create a new file or open a file that you have never opened before in HexEdit and verify that the default settings are used.
