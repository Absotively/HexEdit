Clear History Lists Dialog

 

 

Use this dialog to clear histories and lists that show what files you have been using and what you have been doing.  If other people have access to the same computer you may want to keep this information private.  Note that when files are cleared they are first overwritten with zeroes before being deleted, so that they cannot be recovered using an "undelete" utility.

 

This dialog is invoked using the Clear Histories button on the System Options page.

 

Clear search & replace histories

This clears the search history that appears in the Find Tool (see Tools) and in various pages of the Find dialog.  It also clears the replace history of the Replace page of the Find dialog.

Clear recent file list

This completely clears the file containing the list of recently opened files.  The recent files on the File menu are removed and all entries that appear in the Recently Opened Files dialog disappear.  Note that formating information (character set, font, etc) and highlighting are also deleted when the recent file list is deleted as this information is stored in the same file.

Clear all bookmarks

Clears all bookmarks that appear in the Bookmarks dialog.

Always clear the above on exit

Whenever you exit HexEdit normally all the above items that are checked are deleted.  If HexEdit is terminated abnormally (say with Task Manager) then the lists are not cleared.

Clear

Clears all of the selected lists and exits the dialog..

Help

Displays this information in the on-line help.
