Activation dialog

 

Use this dialog to complete registration of your copy of HexEdit Pro.  You can use this dialog to enter your activation code once you have paid for the software.  It is normally invoked using the Activation button in the About box.

 

Name

Use the same name as you used when you applied to buy HexEdit Pro.  You can vary the case of letters and add spaces and punctuation, but the characters of the name itself must be the same.

Activation Code

Once you have given us your name and paid for HexEdit Pro you will be issued with an Activation Code.  After you enter the nine characters of the Activation Code click the Activate button to complete the registration.

 

Note that to avoid confusion and mistakes the Activation Code is case-insensitive.  Further the letters I and O are interchangeable with (respectively) the digits 1 and 0.

Activate

Click the Activate button once you have entered your Activation Code.  If you have typed the code correctly then you will be licensed to run HexEdit Pro on this machine.  You will then be returned to the About box which has changed to reflect the successful registration.

 

If you typed the code incorrectly then you will be prompted to enter it again.  Double-check your Name, and Activation Code.
